# 📊 Tableau Data Analysis Examples

Repositório com exemplos práticos de análise de dados utilizando **Tableau Public/Tableau Desktop**, com datasets abertos e simulados baseados em cenários reais do Brasil.

---

## 🚀 Projetos Incluídos

| # | Projeto | Dataset | Tipo de Análise | Nível |
|---|---------|---------|-----------------|-------|
| 1 | **E-Commerce Brasil** | `ecommerce_brazil.csv` | Vendas, Regiões, Produtos | Intermediário |
| 2 | **Saúde Pública** | `public_health_brazil.csv` | Série Temporal, Mapas | Intermediário |
| 3 | **Educação & ENEM** | `education_brazil.csv` | Comparativo, Ranking | Iniciante |
| 4 | **Indicadores Econômicos** | `economic_indicators_brazil.csv` | Séries Temporais, Correlações | Avançado |
| 5 | **Transporte & Mobilidade** | `transport_metro_brazil.csv` | KPIs, Dashboard Operacional | Intermediário |

## 🎯 Como Usar Este Repositório

1. **Clone** o repositório:
   ```bash
   git clone https://github.com/seu-usuario/tableau-data-analysis-repo.git
   ```

2. **Escolha um projeto** na pasta `projects/`

3. **Abra o dataset** correspondente no Tableau

4. **Siga as perguntas** em `questions.md` para praticar

5. **Consulte as dicas** em `tableau-tips.md` para técnicas específicas

6. **Publique** seu dashboard no Tableau Public e compartilhe!

## 📊 Exemplos de Dashboards Sugeridos

### 1. E-Commerce Brasil
- Mapa de vendas por estado
- Top 10 produtos mais vendidos
- Análise de métodos de pagamento
- Taxa de cancelamento por categoria
- Série temporal de faturamento

### 2. Saúde Pública
- Mapa coroplético de casos por estado
- Linha do tempo de novos casos
- Taxa de ocupação de UTI
- Comparativo entre regiões
- Correlação entre testes e casos

### 3. Educação
- Ranking de escolas por nota média
- Comparativo público vs privado
- Heatmap de notas por disciplina
- Taxa de aprovação por região
- Correlação infraestrutura x notas

### 4. Indicadores Econômicos
- Série temporal IPCA x Selic
- Correlação Dólar x IBOVESPA
- Desemprego ao longo do tempo
- Comparativo mensal/anual
- Previsão de tendências

### 5. Transporte
- Mapa de calor de passageiros
- Satisfação por linha
- Incidentes vs atrasos
- Comparativo dias úteis vs fins de semana
- KPIs operacionais
